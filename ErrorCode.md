# Error Codes Definition

## BunnyPHP Framework

|Code|Description|
|:---:|---|
|0|ok|
|-1|network error|
|-2|mod does not exist|
|-3|action does not exist|
|-4|template does not exist|
|-5|template rendering error|
|-6|database error|
|-7|parameter cannot be empty|
|-8|internal error|

## Common

|Code|Description|
|:---:|---|
|1|invalid csrf token|
|2|invalid file|
